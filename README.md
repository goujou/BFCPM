## Installation

```
conda create --name BFCPM python=3.9
conda activate BFCPM

git clone --recurse-submodules https://github.com/MPIBGC-TEE/bgc_md2.git
cd bgc_md2
./install_developer_conda.sh

cd ..
git clone https://github.com/goujou/BFCPM.git
cd BFCPM
./install_developer_conda.sh
```
