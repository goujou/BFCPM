allometries_marklund(dbh, H, species, components, custom_species_params)[source]#

Compute tree component biomasses.

Compute tree biomasses for Scots pine, Norway spruce or birch based on [1]. Stump and root biomass is not available in [1] and is taken from [4].

Parameters:

  • dbh (float) – diamter at breast height [cm]

  • H (float) – tree height [m]

  • species (str) – element of [“pine”, “spruce”, “birch”]

  • components (Tuple[str]) – tuple with elements from (“leaves”, “stemwood”, “stembark”, “livingbranches”, “stump”, “roots”)

  • custom_species_params (Dict[str, Dict[str, Any]]) – tree species parameters (for all species)

Returns:

summed biomass [kg_dw] over the components

Notes

Source: [1], [2], [3], [4]

allometries_repola_and_lehtonen(dbh, H, species, components, custom_species_params, x1=None)[source]#

Compute tree component biomasses following Repola and Lehtonen.

Compute tree biomasses for Scots pine, Norway spruce ([6] if dbh < x1 else [5]). For leaves use [7] instead of [6]. Normalize the young curve such that it meets the old curve.

Parameters:

  • dbh (float) – diamter at breast height [cm]

  • H (float) – tree height [m]

  • species (str) – element of [“pine”, “spruce”, “birch”]

  • components (List[str]) – tuple with elements from (“leaves”, “stemwood”, “stembark”, “livingbranches”, “stump”, “roots”)

  • custom_species_params (Dict[str, Dict[str, Any]]) – tree species parameters (for all species)

  • x1 (Optional[float]) – dbh to switch from [6] or [7] to [5], if None use param values: dbh_mean - dbh_sd ([5], Table 3)

Returns:

summed biomass [kg_dw] over the components

Notes

Source: [5], [6], [7]
