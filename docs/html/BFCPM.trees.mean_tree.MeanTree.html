MeanTree#

Qualified name: BFCPM.trees.mean\_tree.MeanTree

CCCC

class MeanTree(name, C_only_tree, z, loc, ctr, management_strategy, base_N_per_m2, status_list, N_per_m2_list, output_fluxes_list=None, cutting_fluxes_list=None, newly_planted_biomass_list=None)[source]#

Bases: object

Class for a MeanTree in a stand.

This is initialized by add_tree() and adds particular stand parameters to the MeanTree. A MeanTree represents N_per_m2 identical trees in the stand.

Parameters:

  • name (str) – name of the MeanTree

  • C_only_tree (SingleTreeCModel) – SingleTreeCModel,

  • z (ndarray) – grid of tree height layer boundaries [m]

  • loc (Dict[str, Any]) –

    location dictionary

    • ”lat”, “lon”: latitude and longitude [deg]

  • ctr (Dict[str, Any]) –

    control dictionary

    • ”phenology”: include phenology?

    • ”leaf_area”: “seasonal LAI cycle?

    • ”water_stress”: include water stress?

  • management_strategy (ManagementStrategy) – instance of ManagamentStrategy

  • base_N_per_m2 (float) – number of trees per ha to be planted usually represented [m2-1]

  • status_list (List[str]) –

    list of tree history, values allowed:

    • ”waiting”: not yet to be planted, combine with planting action

    • ”assigned to: [‘plant’]”: to be planted

    • ”assigned to: [‘wait’, …, ‘plant’]: wait and then pant

  • output_fluxes_list (Optional[List[Dict[Tuple[str, Optional[str]], NewType()(Q_, Quantity)]]]) – history of external output fluxes

  • cutting_fluxes_list (Optional[List[Dict[Tuple[str, str], NewType()(Q_, Quantity)]]]) – history of fluxes caused by cutting

  • newly_planted_biomass_list (Optional[List[Quantity]]) – history of newly planted biomass

  • N_per_m2_list (List[float]) –
