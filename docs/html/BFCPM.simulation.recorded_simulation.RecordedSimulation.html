classmethod from_simulation_run(sim_name, logfile_path, sim_profile, light_model, forcing, custom_species_params, stand, emergency_action_str, emergency_direction, emergency_q, emergency_stand_action_str='', recorded_spinup_simulation=None)[source]#

Run a simulation and record a bunch of variables.

Parameters:

  • sim_name (str) – name of the simulation

  • logfile_path (Path) – path to the logfile

  • sim_profile (List[Tuple[str, float, float, List[Tuple[str, str]], Optional[str]]]) – List of trees with data for the simulation

  • forcing (DataFrame) – external forcing DataFrame

  • stand (Stand) – stand to be used for simulation

  • emergency_action_str (str) –

    element of [“Cut”, “CutWait3AndReplant”, Thin”, “Die”]

    • Cut: Cuts a number of trees from above or below

    • CutWait3AndReplant: Like “Cut” but with delayed replanting.

    • Thin: Thin the entire stand equally.

    • Die: Remove the unsustaibale tree.

  • emergency_direction (str) – above or below, applies only for cutting

  • emergency_q (float) – Fraction of trees to keep on “Thin”

  • emergency_stand_action_str (str) – What to do with remaining trees in stand.

  • recorded_spinup_simulation (Optional[RecordedSimulation]) – If simulation is to be continued from here.

  • light_model (str) –

  • custom_species_params (Dict[str, Dict[str, Any]]) –

Returns:

  • simulation, storing all data

  • additonal variables than can be added to the dataset
